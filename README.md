# kfilechooser
Lightweight Android File Chooser

# Feature
* File and folder chooser mode
* File and folder name and extension filtering
* Easy to use

# Intall

```
repositories {
    maven {
        url  "https://dl.bintray.com/kalibrantas/android" 
    }
}
```


```
compile 'kbrs.com.kfilechooser:kfilechooser:1.0'
```
